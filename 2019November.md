# Novemer Katas 2019

## Beginner's Kata:
Word Count

Create a program that counts the words in a given file

The program should:
 * Accept a text file from the user
 * Count the words in that file


## More Senior Kata:
Grep

   more files that match a user-provided search query (known as the pattern)'
